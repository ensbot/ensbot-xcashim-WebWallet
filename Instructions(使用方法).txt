1.配置钱包节点

有两种方式: 配置官方钱包节点 或 配置本地节点(高级用户)


配置官方钱包节点:
双击webwallet文件夹下的index.html,在浏览器中会进入钱包web界面
点击右上角的Network XCH (unused)下拉列表框,选择其中的"Add XCH Network / Node"

在弹出的 设置您的节点参数 界面中配置如下参数：
NodeName: XCHwallet
URL: http://118.184.66.119
Port: 9888

在下面的单选选项中选择 XCH
然后点击"保存并使用节点"



配置本地节点(高级用户):
双击XCH文件夹下的start.bat
运行一个XCH本地的节点,
运行中如果提示：覆盖<Y是/N否/A全部>? 键盘输入字母 A,然后按Enter。
等待十几分钟，待XCH本地节点数据同步完成后

双击webwallet文件夹下的index.html,在浏览器中会进入钱包web界面
点击右上角的Network XCH (unused)下拉列表框,选择其中的"Add XCH Network / Node"

在弹出的 设置您的节点参数 界面中配置如下参数：
NodeName: localwallet
URL: http://127.0.0.1
Port: 9888

在下面的单选选项中选择 XCH
然后点击"保存并使用节点"



2.以上设置完成后即可使用钱包






1. configure the wallet node
There are two ways: configure official wallet nodes or configure local nodes (advanced users).


Configure the official purse node:
Double click the index.html under the webwallet folder, and enter the wallet web interface in the browser.
Click the Network XCH (Unused) drop-down list box on the upper right corner to select the "Add XCH Network / Node".
The following parameters are set up when setting up your node parameter interface.
NodeName: XCHwallet
URL: http://118.184.66.119
Port: 9888
Select XCH in the following selection options
Then click "save and use the node"


Configuring local nodes (advanced users):
Double click start.bat under the XCH folder
Run a XCH local node,
In operation, if prompted: overlay <Y is /N no, /A all > keyboard input letter A, then press Enter.
Waiting for more than ten minutes, after XCH local node data synchronization is completed.
Double click the index.html under the webwallet folder, and enter the wallet web interface in the browser.
Click the Network XCH (Unused) drop-down list box on the upper right corner to select the "Add XCH Network / Node".
The following parameters are set up when setting up your node parameter interface.
NodeName: localwallet
URL: http://127.0.0.1
Port: 9888
Select XCH in the following selection options
Then click "save and use the node"




2. can be used when the above work is completed
